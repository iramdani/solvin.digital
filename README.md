# Solvin Digital

Solvin Digital is a modern digital agency focused on delivering high-quality web and mobile solutions.

## Getting Started

This project is currently a standalone landing page. You can view it by opening `index.html` in any web browser.

### Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Inter Font (Google Fonts)
- Vanilla JavaScript

- Provide custom web development solutions.
- Design intuitive mobile app interfaces.
- Execute strategic digital marketing campaigns.
